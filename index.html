Architecture

Системийн архитектур

Гол урсгал, төхөөрөмжийн үүрэг, integration цэгүүдийг business-friendly байдлаар тайлбарлав.

Layer 01

Field Node

Галын дохиолол, sensor input, event state болон локал payload-г цуглуулж LoRaWAN packet болгон дамжуулна.

  • ESP32 эсвэл төстэй controller ашиглах боломжтой
  • Alarm, smoke, relay status зэрэг event source-уудтай холбогдоно
  • Battery эсвэл external power дээр ажиллах хувилбаруудтай
Layer 02

Gateway Layer

Node-оос ирсэн packet-үүдийг backhaul network-ээр дамжуулж server тал руу тогтвортой хүргэнэ.

  • RAK төрлийн gateway болон rooftop байршуулалтанд тохиромжтой
  • Multiple node traffic-ийг нэг цэгт төвлөрүүлнэ
  • Coverage өргөжүүлэхэд node update шаардахгүй
Layer 03

Server + Monitoring

ChirpStack, MQTT, webhook, database, dashboard, alerting системүүд энэ шатанд уялддаг.

  • Real-time alert болон event logging
  • Dashboard, report, API integration нэмэх боломж
  • Security, analytics, архив, notification flow-уудыг сервер талд хийнэ
